  18. Next JS Deployment

      19. (For SEO) Deployment of the Next JS needs a bit of knowledge about node js, npm, pm2 technologies. We have assumed that you are using a debian based OS, apt is your package manager. If you are using any other linux distro then apt will be replaced with the respective package manager of the OS


      First zip your whole web code and upload to your server and after unzip your web code and open terminal in that folder and follow this steps.

    1. First you need to install the latest node js version
      sudo apt install nodejs
    2. Check if node js is installed correctly using this command
      node -v
    3. Then you need to install required packages for your system. This will take some time to install. So make sure to not to close this process
      npm install --force
    4. Now you need to install pm2
      sudo npm install -g pm2
    5. Now Create Production optimised build using
      npm run build
    6. now create a auto restarting process using pm2
      pm2 start "npm start" -n "Ebroker Demo Site"
    7. Check if pm2 process is Running OK
      pm2 ls

      When you run pm2 ls you will see , 2 types of output


      1. Error :
      2. Success :

      If you are getting errors in the pm2 process , then run pm2 logs and send us the Screenshot of the error, so that we can guide you to resolve your issues.

      If you want to restart your pm2 process then run pm2 restart id // Replace id with your process id

      For example here id is 0 in the above screenshot --> pm2 restart 0

    8. Create Reverse Proxy for the Node JS server in your apache server settings

      <Location />
      ProxyPass http://localhost:8001/
      </Location>

    9. You needs a bit of knowledge about this reference link of packages and technologies
      Node js : https://nodejs.org/docs/latest/api/
      npm : https://docs.npmjs.com/
      pm2 : https://pm2.keymetrics.io/docs/usage/quick-start/
